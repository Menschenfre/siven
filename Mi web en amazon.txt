+ Pasos a realizar para restablecer la contraseña de MYSQL y ACCEDER a PHPMYADMIN:

1º. sudo systemctl stop mysql.service
2.º sudo mkdir /var/run/mysqld
3.º sudo chown mysql: /var/run/mysqld
4.º sudo mysqld_safe --skip-grant-tables --skip-networking &
5.º sudo mysql -u root mysql
6.º UPDATE mysql.user SET authentication_string=PASSWORD('NUEVOPASSWORD'), plugin='mysql_native_password' WHERE User='root' AND Host='localhost';
7.º EXIT
8.º sudo mysqladmin -S /var/run/mysqld/mysqld.sock shutdown
9.º sudo systemctl restart mysql.service
